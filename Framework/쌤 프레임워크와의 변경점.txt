1. 레퍼런스 카운트 삭제
2. 그래픽 디바이스 필요할때마다 DEVICE 매크로로 사용.
3. 멤버로 그래픽 디바이스 가지고 있었던 것 삭제
4. 랜더 매니저로 변경. 그릴 오브젝트에게 랜더 컴포넌트 붙여주는건 똑같음.
5. 프로토타입 미리 생성 안해줘도 됨. 프로토타입 리스트에 없으면 알아서 새로 등록하고 Clone을 가져옴.
6. 리소스 정보들을 저장하는 리소스 매니저 추가. CTexture 컴포넌트에서 리소스를 경로를 찾지 않음.
7. 레벨 넘어갈 때, 레벨마다 각자 다른 텍스쳐들을 리소스 매니저에 등록함. 프로토타입과 같은 개념
등록 안해줘도 됨. 없으면 알아서 등록하고 Clone을 가져옴.
8. 텍스쳐를 미리 불러올때, MEMORY_TYPE을 정함. 
MEMORY_STATIC: 레벨이 넘어가도 텍스쳐가 삭제되지 않음.(예: 플레이어, UI, 로딩전용 UI 등등)
MEMORY_DYNAMIC: 레벨이 넘어가면 텍스쳐가 삭제됨. 
9. 레퍼런스 카운트가 없어진 대신, 원본 포인터가 아닌 참조 포인터는 Set_WeakPtr(&참조 포인터) 등록한다.
등록해둔 참조 포인터는 원본 포인터가 삭제되면 자동으로 nullptr로 처리된다.
Set_WeakPtr에 등록해둔 포인터는 Safe_Release 하면 안됨. Safe_Release는 원본 포인터만.
10. CTexture 컴포넌트 삭제. Renderer에서 Bind_Texture를 담당함.
